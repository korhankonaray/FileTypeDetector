Installation

The theme can be uploaded in two ways:

  • FTP upload: Using your FTP program, upload the non-zipped Blessing folder into the /wp-content/themes/ folder on your server.
  • WordPress upload: Navigate to Appearance > Add New Themes > Upload. Go to browse, and select the zipped Blessing folder. Hit Install Now and the theme will be uploaded and installed.

Once the theme is uploaded, you need to activate it. Go to Appearance > Themes and activate Blessing theme.

After that you should define either you are going to use one-click-demo-install service to get the website content as we we have on our demo version, or you will build your website on your own wihout using demo installation.

It is very important. If you do not need demo content then after first theme activation go to Theme Options panel and press "Save all changes" button.

One Click Demo Install

We made it so easy for you to get demo content for your website as demo site has. You only need to know, to check and to do few important things:

  1. One-click-demo-install service works only on freshly installed WordPress.

Permission for “cache”/ “timthumb_cache” folder

NOTICE: This information will be useful if you would like this theme works with timthumb service.

Once you install and activate the theme your next step it to set permissions 777 for folder “cache”/ “timthumb_cache” folder to your hosting.

First of all follow this way to find it:

(../wp-content/themes/your-theme-name/timthumb_cache)

Step 1

Please set connection with your server using your FTP client or just make sure it is already connected. Then go to the root directory with the theme.

Step 2

Click on the folder "timthumb_cache" right mouse button and select settings "Change permissions".

Step 3

In the pop-up window of the "Change permissions" please select the settings "Octal notation" 777 as the access rights to the folder. Don't forget to put a tick "Process selected directories recursively". And click "Change" button.

A) Mail Subscriber main function

Mail Subscriber is the application inside the theme, which greatly helps you to notify all and each of your subscribers about further important dates, events, news, meetings, actions, etc.

The main function of this application is to send out notification letters to people who got subscribed to be informed with your web-site latest news. To sign up with this service your visitors can by using of special widget “TheamMakers Newsletters Subscription”.

B) Mail Subscriber in Use

Starting work with Mail Subscriber go to your web-site admin WP panel, and take a look at the left bottom section. Here you see “Mail Subscriber”.

Let's highlight few tips how to work with Mail Subscriber. Before you start edit or create news letters, you should think - about which topics, subjects or themes you would like to inform your customers. Each user after registration receives log in and password to his/her e-mail address to manage their subscriptions: to change the topic of news, or even unsubscribe.

You can split up your news into several groups. So, your first step is to create several e-mail groups for news due to the subjects.

C) Email Groups creation

First, you go to Mail Composer. Here you are able to choose Email Layouts: one column or two columns.

Than goes the list of all themes and topics you created.

You can split up your news into several groups. So, your next step is to create several e-mail groups for news due to the subjects.

To add or create new topic you go to Subscription Groups and create here a new one by putting a title and pressing on “+”.

This new group is displayed in section Mail Composer. Moreover, you can see the certain amount of subscribers near the each of your topics created.

D) Newsletter Text Editing

Also, here you can check how your newsletter looks like. (one column, image below),

two columns, image below

Pay attention that when you already made all changes and need to save them, you should press on the Floppy Icon at the up left corner of this editor.

E) Manage your Mail Subscriber application

Settings section gives you an ability to edit this mail service. Here you can set:

  • User subscribe page id (choose the page id where all users will have an ability to subscribe/unsubscribe from their subscriptions.)
  • Letters per minute (on Web Hosting Hub servers, there is a limit to the number of emails that can be sent per hour (50 messages per domain, per hour). This is for the protection of servers and to help prevent spam. If your mailing list is greater than 50 email addresses, the first 50 messages,for example, would be sent, but the remaining messages would not. You would also not be able to send emails from any email account on that domain for one hour. If you have a need to send more than 50 emails per hour, you can send request to hoster)
  • Set senders mail address (Set an e-mail address from which mails would come to subscribers, by default it is the web-site admin)
  • Set mail header (Set default text for your subscribers which will be shown at the very top, right above your e-mail body)
  • Set mail footer (Set default text for your subscribers which will be shown at the very bottom, right below your e-mail body)
  • New user registration text (Set default text for your subscribers after subscription)

Please make sure that after each editing you save all changes!

F) Newsletters Sending

When all changes are made and you are ready to send these newsletters to your subscribers, go back to Mail Composer, mark email groups (it's very important, cause if do not mark – letters will not be sent).

So, you make sure that e-mail groups needed are marked, next press “Send letter” button.
Than you will be asked to confirm the sending. When you confirmed it, the Progress Bar appears in front of you, where you can see the sending process.

G) History

Mail Subscriber application memorizes all letters sent to your subscribers.

It helps if you need to change some title, words, dates, not all information, but some of it. You can pick one letter from History and in the same way to edit it, make some changes, and send.

If for some reasons your subscriber gets some troubles with sign up, log in, or editing his/her subscription, you can do it on your own. Go to Users in WP admin panel, choose one user and do actions requested.
